# Memester v0.1.0

First public release of Memester, a dependency-free client-side meme generator built with vanilla HTML, CSS, JavaScript, and Canvas.

## Highlights

- Browse a built-in gallery of meme templates and upload custom images.
- Create and edit multi-line memes on canvas with font, color, size, alignment, and sticker controls.
- Save memes locally with `localStorage` and download finished images as JPG files.
- Ship a cleaner public repository baseline with license, security policy, changelog, validation tooling, CI checks, and improved project documentation.
- Keep public-demo sharing disabled by default until a restricted Cloudinary preset is explicitly reviewed.

## Repository and documentation updates

- Added lightweight static validation with `npm run validate`.
- Added a GitHub Actions static-quality workflow for push and pull request checks.
- Expanded the README with project status, setup guidance, screenshots, validation expectations, and release policy details.
- Organized runtime media and documentation assets under `assets/`.

## Notes

- Release tag format remains `vMAJOR.MINOR.PATCH`.
- This release does not enable any secret-backed or server-side sharing flow.
